# bluzelle_dart

A library for Dart developers.

To generate Protobuf file implementations => `./scripts/generate_proto.sh`

## Usage

A simple usage example:

```dart
import 'package:bluzelle_dart/bluzelle_dart.dart';

main() {
  var awesome = new Awesome();
}
```
